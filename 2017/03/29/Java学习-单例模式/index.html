Java学习-单例模式

2017-03-29
Java

目标

  • 设计一个类，只能生成该类的一个实例
  • 单例模式

关键点

  • 一个类只有一个实例，这是最基本的
  • 它必须自行创建这个实例
  • 它必须自行向整个系统提供这个实例

实现方式

  • 懒汉模式（类加载时不初始化）
  • 饿汉模式（在类加载时就完成了初始化）
